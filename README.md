#### Setup/Installation

Copy-paste this into your terminal:

```sh
git clone https://github.com/mkdirlove/ExpenseTracker.git
```
```
cd ExpenseTracker
```
```
python3 -m pip install -r requirements.txt
```
or
```
python3 app.py
```

#### You can now visit your local webserver link: http://127.0.0.1:5000/
